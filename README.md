<h1>Documentacion del proyecto</h1>
<h2>Descripcion: </h2>
<p>El proyecto seleccionado es una pagina informativa con objetivo de una pagina de un club de hanball llamado SAPA con ubicacion en Marcos Paz. Una página informativa es un sitio o sección web diseñada específicamente para transmitir datos, conocimientos o detalles sobre un tema, entidad o servicio de manera clara, estructurada y accesible. Su objetivo principal no es vender directamente (como un e-commerce), sino educar, informar o resolver dudas del usuario. Permitirá a los usuarios poder ver fixtures, completar un formulario y proximos partidos asi como tambien ver las redes sociales del equipo y llevarlos a dichas redes.</p>
